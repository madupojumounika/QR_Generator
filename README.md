# 📱 QR Code Generator

A simple and responsive **QR Code Generator** built using **HTML, CSS, and JavaScript**.  
This project allows users to generate QR codes instantly by entering any text or URL.

---

## 🚀 Features

- Generate QR codes instantly
- Supports both text and URLs
- Clean and simple UI
- Responsive design
- Attractive gradient heading

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- QRCode.js (CDN)

---

## 📂 Project Structure

```
QR-Code-Generator/
│── index.html  
│── style.css  
│── README.md  

```
---

## ⚙️ How It Works

1. Enter text or URL in the input box  
2. Click on **Generate QR Code**  
3. QR code will be generated instantly  

---

## 🧑‍💻 Installation & Usage

1. Clone the repository:
   git clone https://github.com/your-username/qr-code-generator.git

2. Navigate to the folder:
   cd QR_Generator

3. Open the project:
   - Open `index.html` in your browser
